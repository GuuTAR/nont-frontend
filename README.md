# nont-frontend

# Project installation
1. git clone
2. git branch dev/{new-branch-name} (ex. git branch dev/login-page)
3. npm install
4. npm start
  
# Feature commit
1. git checkout {your-branch-name}
2. git add .
3. git commit -m {commit message} (ex. git commit -m "Add login page")
4. git push
